Continously Improving Terrain Model
The best available terrain information is processed to create a hydrologically correct moedel that can be used for a variety of applications.
Elevation models available from NASA and the USGS are saved as raster images. Where available, LIDAR data is used to create locally high resolution models. Additionally, hydrologic vector features are used to further process the model to create a hydrologically correct elevation model. The result is a world wide coverage digital elevation model that is continuously being refined and is served as a web map tile service. The DEM can be used for a variety of applications including disaster response, watershed analysis, and preliminary infrastructure design.

Terrain Based Applications
Preliminary infrastructure design is possible with the digital terrain map maintained by Open Terrain Map. The first design generation algorithms are being built for gravity fed drinking water supply systems and low bore sanitary sewer systems.
Where the elevation model does not provide adequate resolution, the best available model is resampled using a spline technique to allow for a reasonably high resolution with which to produce a preliminary design for small scale infrstructure projects. The idea was focused on projects in the developing world but it would likely be useful rural and small town projects as well. The goal is to take the focus off the technical aspects of infrastructure design and put the focus on the more important aspects for the sustainability of the project which is the financial and logistic orgnaization of the project.
